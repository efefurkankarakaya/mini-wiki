2 Attributes $().attr(), $().prop(), $().css() var node = $("a").attr(href); -> gets a tags' href values
$("a").attr(href, "https://github.com/efefurkankarakaya"); -> sets the url to a tags' href attribute
$("a").attr(class, "test_class"); -> sets the new class name to all a tags
$("#check-box").prop("checked"); -> sets checked prop value true
$("#box").css("background-color"); -> gets background-color value
$("#box").css("background-color", "blue"); -> sets blue to background-color value
